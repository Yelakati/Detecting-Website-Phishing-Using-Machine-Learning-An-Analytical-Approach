About the Phishing Detection System

Website Phishing Detection using Extreme Learning Machine

What is Phishing?

Phishing is a type of cyber attack where attackers create fake websites that mimic legitimate ones to steal sensitive information such as usernames, passwords, credit card details, or other personal information. These attacks often come through emails, social media messages, or other communication channels that direct users to these malicious websites.

Our Approach

This application uses an Extreme Learning Machine (ELM) model to detect phishing websites. ELM is a fast learning algorithm for single-hidden layer feedforward neural networks that randomly assigns input weights and analytically determines output weights.

The model analyzes various features of URLs and websites, including:

  • URL structure and components
  • Domain characteristics
  • Security indicators
  • HTML and JavaScript elements
  • External links and resources

Advantages of ELM for Phishing Detection

  • Speed: ELM's training is extremely fast as it doesn't require iterative tuning of weights.
  • Performance: Despite its simplicity, ELM achieves comparable performance to more complex algorithms.
  • Scalability: ELM can be applied to large datasets efficiently.

Model Accuracy

Our ELM model was trained on a comprehensive dataset of legitimate and phishing websites. The model achieves:

  • Accuracy: ~94%
  • Precision: ~95%
  • Recall: ~93%
  • F1 Score: ~94%

Note: Actual performance may vary based on real-world usage and evolving phishing techniques.

How to Use the Tool

URL Check: Simply enter a website URL in the input field on the home page and click "Check Website" to get an instant analysis.

Advanced Mode: For technical users, the application also provides an option to manually input website features for analysis.

Disclaimer

While this tool provides reasonably accurate phishing detection, it should not be the sole decision-making factor for determining website legitimacy. Always practice safe browsing habits and use multiple verification methods for sensitive websites.
